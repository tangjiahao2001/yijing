## 易经算卦程序
### 使用方法 USE METHOD
整个运行过程很简单
* 下载文件
```shell
git clone https://github.com/tangjiahao2001/yijing.git
```
* 执行程序，注意python需要是python3以上版本
```python
python yijing.py
```
